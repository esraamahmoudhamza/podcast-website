# podcast-website
"podcast site with HTML, CSS, JS, featuring audio/video content &amp; premium access."

I developed a podcast website using HTML, CSS, and JavaScript. The platform featured two types of content: audio-only podcasts and video podcasts, covering diverse topics such as programming, healthcare, and philosophy. The site included a user-friendly interface, allowing visitors to browse and stream episodes seamlessly. Additionally, I implemented a payment section to support premium content, ensuring a secure and efficient transaction process. This project enhanced my front-end development skills and provided hands-on experience in building functional and visually appealing web applications.
